UITableView

凡是遵循 UITableVIewDataSource 协议的OC对象，都可以是UITableView的数据源。

TableView样式

UITableView拥有两种样式，默认为UITableViewStylePlain

两种样式均可分组，是否分组是根据numberOfSectionsInTableView:的返回值是否 大于1 来决定的。

TableView常用属性

rowHeight 统一设置每行的高度
tableHeaderView 设置TableView整体头部的View，常用于制作图片轮播
tableFooterView 设置TableView底部的显示View，常用于制作“加载更多”按钮
separatorColor 修改TableView的分割线颜色
separatorStyle 修改TableView的分割线样式

UITableView索引方法

返回一个数组，作为索引显示文本
索引是根据组的下标访问，所以和返回值并没有关系。

- (UITableViewCell *)tableView:(UITableView *)tableView cellForRowAtIndexPath:(NSIndexPath *)indexPath;

UITableViewCell

默认有以下几个样式：

typedef NS_ENUM(NSInteger, UITableViewCellAccessoryType) {
UITableViewCellAccessoryNone, // don’t show any accessory view 什么都不显示
UITableViewCellAccessoryDisclosureIndicator, // regular chevron. doesn’t track 显示>按钮
UITableViewCellAccessoryDetailDisclosureButton TVOS_PROHIBITED, // info button w/ chevron. tracks 显示 > 和 感叹号
UITableViewCellAccessoryCheckmark, // checkmark. doesn’t track 会打钩
UITableViewCellAccessoryDetailButton NS_ENUM_AVAILABLE_IOS(7_0) TVOS_PROHIBITED // info button. tracks 仅显示一个感叹号
};

滚动→需要Cell→查找缓存池→无→新建→多余的Cell放入缓存池
↘有→复用↗

UITableViewCell *cell = [tableView dequeueReusableCellWithIdentifier:@”Cell标识”];
//如果返回nil，则表示缓存池中没有可用cell
if (cell == nil) {
//那么则用alloc init创建一个新的Cell，并使用 reuseIdentifier:添加Cell标识
cell = [UITableViewCell alloc] initWithStyle:Cell样式 reuseIdentifier:Cell标识
}
